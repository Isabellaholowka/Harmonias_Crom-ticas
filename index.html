O que é o Círculo Cromático?

O círculo cromático é uma representação visual das cores organizadas de acordo com sua relação entre si. Ele é usado para entender como as cores interagem e quais combinações funcionam melhor no design.

Tipos de Harmonias Cromáticas

  • Monocromática: Usa variações da mesma cor.
  • Análoga: Usa cores vizinhas no círculo cromático.
  • Complementar: Usa cores opostas.
  • Tríade: Usa três cores equidistantes.
  • Tetrádica: Usa duas duplas de cores complementares.

Como Escolher as Cores do Site?

Para manter a harmonia no design, é importante escolher cores adequadas para cada elemento:

  • Cabeçalhos: Azul (#0073E6)
  • Texto principal: Preto ou cinza escuro
  • Links e bordas: Azul-esverdeado (#00B3A6)
  • Botões: Verde-azulado (#0099CC)
